导入流程：
  1. 从飞书/文档获取内容
  2. 调用 Claude AI 提取知识（实体、关系、规则）
  3. 将提取结果提交到审核工作流
  4. 在审核页面批准/拒绝候选条目

⚙️ 飞书配置

飞书导入功能需要在配置中设置以下参数：

  • feishu_app_id - 飞书应用 ID
  • feishu_app_secret - 飞书应用密钥
  • feishu_base_url - 飞书 API 基础 URL
  • anthropic_api_key - Claude API 密钥
